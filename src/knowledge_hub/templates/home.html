Automation-First Flow

1 Send JSON

Paste or POST a structured session log. The importer can auto-create the project from its slug.

2 Store the truth

Knowledge Hub saves the normalized log, raw JSON, derived blockers, next steps, and project timeline.

3 Generate context packs

Use the API or CLI to build a ready context pack for the next chat without repeating yourself.
